# Content for [Beyond Semicolon](https://beyondsemicolon.com)

## How does it work?
1. Create a new directory (with unique slug) for every new code snippet that you want to show.
2. Visit `code-for-ghost` project's domain and get your unique `iframe URL` (or `ID` to be used with the injected script).
3. Paste that iframe code on your blog.
